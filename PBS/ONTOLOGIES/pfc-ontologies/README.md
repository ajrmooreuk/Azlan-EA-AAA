## OAA Compliance

All PFC ontologies target OAA v5.0.0 compliance with 7 gates:
- G1: Namespace/Context
- G2: Relationship Cardinality
- G2B: Entity Connectivity
- G2C: Graph Connectivity
- G3: Business Rules
- G4: Documentation
- G5: Test Data
- G6: Registry Registration
