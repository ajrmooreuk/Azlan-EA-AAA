# PE-PPM-ONT (Portfolio/Programme/Project Management)

**Current Version:** 3.0.0
**OAA Schema Version:** 5.0.0
**Status:** Production
**Part of:** PE-Series-ONT

## Overview

Portfolio, Programme, and Project Management ontology. Provides the execution layer for organizational initiatives, connecting strategic objectives to deliverable work packages via PBS (Product Breakdown Structure) and WBS (Work Breakdown Structure).
